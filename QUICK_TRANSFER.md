# 🚀 Quick Transfer Guide – BookNest

Transfer this project to another device in 3 simple steps.

---

## Step 1: Transfer Files

Choose one method:

### Method A: Using Git (Recommended)
```bash
git clone https://github.com/your-username/booknest.git
cd booknest
```

### Method B: ZIP Archive
1. Zip the `booknest/` folder
2. Transfer via USB/Cloud/Email
3. Extract on target device
```bash
cd booknest
```

### Method C: Cloud Storage
1. Upload `booknest/` to Google Drive / Dropbox / OneDrive
2. Download on target device
3. Extract and navigate

---

## Step 2: Run Setup

### Windows
Double-click: `setup.bat`

Or manually:
```bash
npm run install:all
```

### macOS/Linux
```bash
chmod +x setup.sh
./setup.sh
```

Or manually:
```bash
npm run install:all
```

---

## Step 3: Start the Project

### Automatic (Easiest)
```bash
npm run dev  # Terminal 1
node scripts/local-api-server.cjs  # Terminal 2
```

### Manual
**Terminal 1:**
```bash
node scripts/local-api-server.cjs
```

**Terminal 2:**
```bash
cd frontend && npm run dev
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Node not found | Install from https://nodejs.org/ |
| Port 3001 in use | Change port in `local-api-server.cjs` |
| Module errors | Run `npm run install:all` |
| Frontend blank | Ensure API runs on port 3001 |
